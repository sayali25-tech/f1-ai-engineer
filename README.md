# F1 AI Race Engineer

AI-powered Formula 1 race strategy dashboard built with Python and Streamlit.

## Features

- Live F1 telemetry dashboard
- Tyre degradation prediction
- Pit stop strategy optimizer
- Monte Carlo race simulation
- Driver pace comparison
- Interactive track map visualization
- Race strategy analysis tools

## Tech Stack

- Python
- Streamlit
- FastF1
- Pandas
- NumPy
- Matplotlib

## Future Improvements

- Real-time race prediction
- Weather integration
- AI-based overtaking prediction
- Team strategy comparison
